# Minigrep

https://doc.rust-lang.org/book/ch12-00-an-io-project.html

1. Close repo
   ```
   git clone https://github.com/mshandruk/minigrep.git
   ```
2. Run tests
   ```
   cargo test
   ```
3. Build
    ```
    cargo build --release
    ```

4. Usage
   ```
   target/release/minigrep <word> <file>
   ```
